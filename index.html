Bem-vindos ao projeto Resistence.


O nosso projeto tem como função realizar os cálculos dos circuitos solicitados, para facilitar a utilização do usuário separamos em Eletrônica Industrial e Sistemas Digitais, cada um com as devidas especificações.


Para acessar a página de Sistemas Digitais, basta utilizar o menu e selecionar 'Sistemas Digitais'.


Para acessar a página de Eletrônica Industrial, basta utilizar o menu e selecionar 'Eletrônica Industrial'.


Esperamos que você tenha uma ótima experiência e conte conosco para te auxiliar da melhor maneira possível.
